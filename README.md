# Screen Recorder
누구나 무료로 쓸 수 있는 Screen Recorder


모듈 설치: 
  pip3 install numpy opencv-python pyautogui keyboard


팁: 
  1. r_ctrl과 l_ctrl을 동시에 눌러 정지.
  2. 촬영 파일은 .avi 파일로, 영상 촬영기와 같은 디렉토리에 저장.
  3. [Credits]에 있는 코드를 개조해서 만듦.
